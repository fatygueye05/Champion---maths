# Champion des Maths

## Description

Ce projet est un programme développé en langage C permettant de s’entraîner aux mathématiques à travers plusieurs mini-jeux.
L’utilisateur peut choisir différentes opérations (addition, soustraction, multiplication, etc.) et obtenir un score en fonction de ses réponses.

## Fonctionnalités

* Addition de nombres aléatoires
* Soustraction avec résultat positif
* Multiplication
* Tables de multiplication
* Gestion des scores
* Sauvegarde des résultats dans un fichier

## Organisation du projet

* `src/` : contient le code source du programme (`main.c`)
* `README.md` : description du projet

## Prérequis

* Un compilateur C (GCC)
* Un environnement de développement (Code::Blocks, VS Code…)

## Compilation

Pour compiler le programme, utilisez la commande suivante :
gcc src/main.c -o jeu

## Exécution

Après compilation, lancez le programme avec :
./jeu

## Utilisation

1. Lancer le programme
2. Choisir une opération dans le menu
3. Répondre aux questions affichées
4. Consulter son score
